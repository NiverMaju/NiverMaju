# Niver da Maju - Orkut Retrô

## Como colocar no GitHub Pages
1. Abra o repositório NiverMaju/NiverMaju no GitHub.
2. Apague arquivos antigos de teste, principalmente index_FINAL_Niver_Maju.html e index(1).html.
3. Envie TODO o conteúdo desta pasta para a raiz do repositório.
4. A raiz precisa ter: index.html, style.css, script.js, firebase.js, fotos/, avatares/.
5. Aguarde 1 minuto e aperte Ctrl + F5 no site.

## Firebase
Cole as regras que estão em REGRAS_FIRESTORE.txt em Firestore Database > Regras.

## Comentários
Os scraps ficam salvos na coleção `scraps` do Firestore e aparecem para todos em tempo real, sem login.
